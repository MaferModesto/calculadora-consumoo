O projeto Calculadora de Consumo é um aplicativo mobile desenvolvido em Java e XML como parte das atividades acadêmicas, com o objetivo de aplicar conceitos de lógica de programação e desenvolvimento Android. A aplicação permite ao usuário inserir dados e calcular o consumo de forma rápida e prática, auxiliando no controle de gastos.
Durante o desenvolvimento, foram trabalhados conceitos como manipulação de entrada de dados, validações, lógica de cálculo e construção de interfaces utilizando XML. O projeto contribuiu para o aprendizado na criação de aplicações mobile nativas, além de fortalecer a organização do código e a experiência do usuário.
